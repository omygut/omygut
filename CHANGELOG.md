# Changelog

## [0.1.11] - 2026-04-15

### 新增
- 时间选择器使用5分钟间隔 (#140)
- 化验指标数据优化及异常检测改进 (#137)
- 历史页面记住分类筛选状态 (#134)
- 日历支持滑动手势和年份导航 (#129)
- 化验记录显示检测分类而非指标数量 (#121)

### 修复
- 修复 app.config.ts 中 cloud 属性类型错误 (#136)
- 化验指标添加缺失的单位字段 (#135)
- 检查记录限制为1张图片 (#123)
- 添加 miniprogram-ci 依赖 source-map-js (#120)

### 性能
- 从详情页返回时跳过重复数据加载 (#130)

### 样式
- 更换症状图标 (#124)

## [0.1.7] - 2026-04-14

### 新增
- 日历弹窗组件，支持在所有表单页面选择日期

### 变更
- 记录页拆分为"日期"和"历史"两个独立页面
- 抽取 RecordItem 组件，统一记录项展示样式
- 表单提交按钮改为跟随内容滚动
- 检查类型选择改为横向按钮样式
- 马赛克工具从框选模式改为画笔模式
- 用药选择改为标签样式，与饮食页面一致
